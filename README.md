##  Como rodar o projeto ?

1. Para rodar o projeto é necessario ter o [Node.js](https://nodejs.org/en/download) instalado.
2. No prompt de comando executar o seguinte comando:

### Entrada
```bash
# Instalar dependencias do projeto.

npm install 
# ou
yarn install
```

### Saída Esperada
```bash
added 360 packages, and audited 361 packages in 3s

128 packages are looking for funding
  run `npm fund` for details

found 0 vulnerabilities
```

3. No prompt de comando executar o seguinte comando:
 
 ### Entrada
```bash
# Rodar servidor local da aplicação.

npm run dev
# ou
yarn run dev
```

### Saída Esperada
```bash
> open-ran@0.1.0 dev
> next dev

   ▲ Next.js 14.1.0
   - Local:        http://localhost:3000

 ✓ Ready in 1350ms
```

4. Acesse o seguinte link http://localhost:3000 para ter acesso a aplicação.
